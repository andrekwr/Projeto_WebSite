Projeto autônomo prático-teórico

Projeto final de Design de Sotware

Em Design de Sotware aprendemos conceitos básicos de algoritimos, fundamentos de Programação, linguagens de programação e desenvolvimento de programas. No projeto final da matéria foi pedido aos alunos que fizessem um projeto completo de software, usando as bibliotecas pygame, flask ou Tkinter da linguagem phyton. O pygame é definido como um "motor de jogo" ou seja, é uma biblioteca usada para a programação de jogos. O flask é um "framework web", um software designado ao desenvolvimento de sites e o Tkinter é uma biblioteca que permite o desenvolvimento de interfaces gráficas.

Projeto final de Ciências dos Dados

Em ciências dos dados, o objetivo da matéria é aprender a elaborar análises exploratórias de dados (univariadas e multivariadas), utilizando ferramentas estatísticas e computacionais, selecionar informações de bancos de dados, tratá-los e prepará-los para processamento, especificar as distribuições de probabilidades adequadas para as variáveis quantitativas discretas e contínuas, conduzir testes inferenciaisadequados que possam dar base à tomada de decisão e analisar relações entre as variáveis, utilizando ferramentas estatísticas inferenciais adequadas. Durante o curso são propostos projetos para demonstrar esses conhecimentos adquiridos.

Projeto final de Modelagem e Simulaçao

Em modelagem e simulaçao do mundo fisíco aprendemos sobre modelagem matematica discreta e contínua de sistemas dinâmicos de 1° e 2° ordem. O foco do projeto final de Modelagem e Simulaçao foi a modelagem de sistemas mecânicos, ou seja, o estudo do movimento de corpos macroscópicos, os modelos foram compostos de sistemas de equações diferenciais modelados por diagramas de corpo livre, que foram ensinados no percorrer do curso.

Projeto final de Co-Design de Aplicativos

A matéria de Co-Design de aplicativos tem como objetivo analisar usuários a partir de entrevistas e hipoteses, sintetizar questões de projeto a partir de análises de usuários, criar conceitos de aplicativos e aplicar ferramentas de design gráfico e tecnologias a partir de front-end web em prototipação digital.
